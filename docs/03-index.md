1. **個化飲食計劃**：根據個人的代謝率、健康狀況和基因設計飲食方案。
2. **均衡營養攝取**：確保足夠的蔬菜攝取以提供維生素和礦物質，避免因過度限制某一種 nutrient 而導致失衡。
